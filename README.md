## 📌 About This Repo

This repository is my personal **day-by-day DSA (Data Structures & Algorithms)** practice log. Each folder corresponds to one day of consistent problem solving. The goal is to build problem-solving muscle memory through daily practice.

- 📅 **One folder per day** — each named `Day_XXX`
- 💡 **Each file** contains a clean C++ solution
- 🔄 **Updated daily** — new problems added every day
- 🎯 **Topics covered** — Arrays, Strings, Binary Search, Bit Manipulation, and more
